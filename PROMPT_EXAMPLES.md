# AI Agent Prompt Examples

To interact with the Antimomentum Beta AI Agent, you can use the `/api/agent/prompt` endpoint. Here are some example prompts you can use to test the capabilities:

### 1. Research & Information Retrieval
**Prompt:**
`"Find the most recent developments in Quantum Computing for early 2026 and summarize how they might affect encryption standards."`

**Context:**
`"The user is a research student looking for advanced technical summaries."`

---

### 2. Code Generation & Calculation
**Prompt:**
`"Write a Python function that calculates the Fibonacci sequence up to N terms using recursion, and then explain its time complexity."`

**Context:**
`"The user is a beginner developer learning algorithms."`

---

### 3. Design & UI Components
**Prompt:**
`"Generate a clean, modern SVG icon representing a universal workspace for a dark-themed IDE."`

**Context:**
`"The IDE uses Tailwind CSS and Radix UI primitives."`

---

### 4. "Formation" (Auto-gen Docs/Code)
**Prompt:**
`"Generate a comprehensive API documentation template in Markdown for a workspace-based IDE that includes authentication and Docker-based code execution."`

**Context:**
`"Antimomentum Beta project documentation."`
